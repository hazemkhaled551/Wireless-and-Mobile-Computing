# 2) Define Mobile Computing.

## Answer:

Mobile Computing refers to the use of computing devices while moving from one place to another without losing network connectivity.

It is an umbrella term used to describe technologies that allow users to access services:

* Anytime
* Anywhere
* Anyplace

Mobile computing also refers to moving work processes from fixed environments into dynamic mobile environments where tasks can be performed remotely.

---

# 3) Explain the different definitions of Mobile Computing.

## Answer:

Mobile Computing has several definitions:

### 1. Computing While Moving

Using a computer or mobile device while moving from one location to another.

### 2. Dynamic Work Environment

Moving work processes from fixed locations into dynamic and flexible locations.

### 3. Performing Tasks Remotely

Allowing tasks to be completed in locations where computing was not previously possible.

### 4. Service Accessibility

Enabling users to access services anytime and anywhere using mobile technologies.

# 7) Compare between Wired Networks and Wireless Networks.

## Answer:

| Wired Networks             | Wireless/Mobile Networks        |
| -------------------------- | ------------------------------- |
| High bandwidth             | Low bandwidth                   |
| Stable bandwidth           | High bandwidth variability      |
| Low delay                  | Higher delay                    |
| Connected operation        | Disconnected operation possible |
| High-resource machines     | Low-resource machines           |
| High power devices         | Low power devices               |
| Physical access required   | Requires proximity              |
| Easier security monitoring | Hidden terminal problem         |

Wireless networks are more flexible and mobile, while wired networks provide higher stability and performance.

# 10) Explain the Seven Waves of Mobile Computing Evolution.

## Answer:

The evolution of mobile computing passed through seven major waves:

## 1. Portability

Reducing hardware size to create movable computers.

## 2. Miniaturization

Creating smaller devices usable while moving.

## 3. Connectivity

Allowing wireless online communication.

## 4. Convergence

Combining multiple devices into one hybrid device.

## 5. Divergence

Creating specialized devices with specific functions.

## 6. Applications (Apps)

Developing mobile applications for entertainment and productivity.

## 7. Digital Ecosystems

Integrating mobile systems with cloud services, social media, and internet technologies.

These waves shaped modern mobile technology.

# 14) Explain Convergence in mobile computing.

## Answer:

Convergence refers to integrating different digital devices into one hybrid device.

For example, a smartphone combines:

* Camera
* Music player
* Gaming device
* Mobile phone
* Video streaming device

This reduces the need for multiple separate devices.

---

# 15) Explain Divergence in mobile computing.

## Answer:

Divergence is the opposite of convergence.

Instead of creating general-purpose devices, divergence focuses on creating specialized devices designed for specific tasks and functionalities.

# 23) Explain intermittent connectivity in mobile computing.

## Answer:

Intermittent connectivity occurs when mobile devices temporarily lose network access.

Causes include:

* Movement between networks
* Weak wireless signals
* Lack of infrastructure

Challenges include:

* Data loss
* Packet loss
* Communication delays

Solutions include:

* Data buffering
* Efficient transfer protocols
* Power-aware communication systems

These ensure reliable communication despite unstable connectivity.

# 24) Why are mobile devices energy constrained?

## Answer:

Mobile devices are energy constrained because:

* They rely on batteries.
* They have compact sizes.
* They use power-intensive sensors and communication systems.
* They lack advanced cooling systems like fans.

Excessive power consumption affects:

* Device temperature
* Battery life
* User experience

Therefore, energy management is a critical design issue.
